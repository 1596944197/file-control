# files-control

mh33.top的子应用，文件云存储

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
pnpm dev

cd service && pnpm dev
```

### Type-Check, Compile and Minify for Production

```sh
// edit docker-compose.yml and then
node start.mjs
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
